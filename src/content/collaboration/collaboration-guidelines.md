## 创作方式
&emsp;&emsp;由于本项目的 wiki 文件由独立的 markdown 文件组成，故对于每一小节，内容贡献者将文章编辑为一个 markdown 文件即可。请注意：**所有需要跳转的内容，在文章中标明“从哪里跳转到哪里”即可（即使您想跳转的内容还并没有创作出来，我们后续会有计划的进行创作），开发者后续会统一进行处理**

## 推荐创作工具与工作流程

&emsp;&emsp;推荐使用 Visual Studio Code 进行创作，并安装 **Markdown Math** 插件以支持数学公式的编辑与渲染。

### 工具介绍

- **Visual Studio Code(以下简称为 VS Code)**：一款功能强大的集成开发环境 **(IDE)** ，支持多种编程语言和文件格式，提供丰富的插件生态系统。在本场景中，VS Code 将作为主要的 Markdown 编辑器，帮助内容贡献者高效地编写和管理 wiki 内容。

- **Markdown Math 插件**：一款专为 VS Code 设计的插件，提供对 Markdown 文件中数学公式的支持。该插件允许内容贡献者使用 LaTeX 语法编写数学公式，并在编辑器中实时预览渲染效果，极大地提升了数学内容的创作效率和准确性。

### 工作流程
1. **安装 VS Code**：访问 [Visual Studio Code 官网](https://code.visualstudio.com/) 下载并安装适用于您操作系统的版本。
2. **安装简体中文插件**：如您需要，可安装简体中文插件进行汉化。
3. **安装 Markdown Math 插件**：打开 VS Code，进入扩展市场，搜索 "Markdown Math" 并安装该插件。
4. **创建 Markdown 文件**：根据需要创建新的 Markdown 文件，命名应清晰且具有描述性，以便于后续管理和查找。
5. **编写内容**：在 Markdown 文件中编写 wiki 内容。点击 VS Code 右上角的“打开侧边预览”按钮，或先同时按下 `Ctrl + K` ，再按 `V` 即可在编辑器旁边实时预览 Markdown 内容的渲染效果，确保格式和排版符合预期。
6. **数学公式**：使用 LaTeX 语法编写数学公式，Markdown Math 插件将自动渲染并显示预览效果。行内数学公式使用 `$...$` 包裹，块级数学公式使用 `$$...$$` 包裹。您可以使用 ai 辅助编写 Latex 代码，或参考这篇文章：[LaTeX 数学公式大全](https://www.luogu.com.cn/article/1gxob6zc)。


## 创作规则

### 内容组成

1. 首先，您应当明确，您所创作的内容属于五大板块中的哪一个板块（详情可见**考点汇总与解析**中的左侧导航栏），并在文章开头标明，便于开发者对文章进行分类
2. 如果您认为，您所创作的内容需要前置知识，需要在文章的开头添加一段声明，例如：
> 前言
>
> 在阅读本节内容前，请确保您已掌握了以下知识点：
> - 相关知识点1
> - 相关知识点2
> - ...
> 
> 如您并未完全掌握上述知识，建议您点击上述知识点，跳转到对应的章节进行学习，以便于更好地理解本节内容。
3. 为了便于用户快速练习，创作者需要在每篇文章的最后附上考题链接，即本篇文章所对应的**所有**题库的考题。示例如下：
> 考题链接
> - MC2-0001
> - MC2-0002
> - ...

### 基本规范

- 请正确使用全角中文标点符号。特别地，句末要有句号。
- 数学公式（运算式、运算符、参与运算的常数、作为变量的字母等）应使用 LaTeX，非数学公式（一般英文单词、题目名、人名等）不应使用 LaTeX。
- 中文与英文、数字或公式之间以半角空格隔开，但中文标点符号与英文、数字或公式之间不应有空格。

### 排版
- 应使用 markdown 正确排版。
- 应使用 `#, ##, ###, ####` 符号表示标题行。标题应对文章结构进行引导；不应滥用标题行表示强调与无意义内容。
- 应使用 `-`, `+`, `*` 来表示无序列表，用 `1.` 来表示有序列表。
- 应使用行内代码块表示字符串或代码，如 `aabc`。

- 应使用行间代码块引用代码。

- 应使用 `[]()` 引用链接。

- 应使用 `![]()` 引用图片。

- 应使用 `>` 引用文本。例如可以引用例题。

### 数学公式
- 数学公式（运算式、运算符、参与运算的常数、作为变量的字母等）应使用 LaTeX。**同一个数学公式应写在一个 LaTeX 环境内。**
- 数学公式中的文本应使用 `\text`，字符串应使用 `\mathtt`。如 $a \text{ is prime}$（`$a \text{ is prime}$`），$S = \mathtt{aabcd}$（`$S = \mathtt{aabcd}$`）。
- 公式独立成行时应使用块级公式。以下是一个块级公式的例子：

```markdown
$$a + b = c$$
```

$$a + b = c$$
