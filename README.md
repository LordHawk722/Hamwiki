# Hamwiki

中国业余无线电操作能力验证考试考点 Wiki。

## 项目说明

这是一个基于 React + Vite 的 Web 端 Wiki 原型，包含：

- 左侧知识点目录
- 关键词搜索
- 分类筛选
- Markdown 内容展示

目前内置了业余无线电考试常见知识点示例，方便你后续持续补充。

## 技术栈

- React 18：用于构建前端界面
- Vite 5：用于本地开发与生产构建
- React DOM：用于浏览器端渲染
- react-markdown：用于渲染知识点 Markdown 内容
- 原生 CSS：用于页面样式与响应式布局

## 本地运行

1. 安装 Node.js 18+。
2. 在项目根目录执行：

	npm install

3. 启动开发服务器：

	npm run dev



## 构建产物

执行：

npm run build

构建结果在 dist 目录。
